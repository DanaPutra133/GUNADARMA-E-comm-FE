# GUNADARMA E-comm FE

Aplikasi e-commerce Gunadarma. Website ini terdiri dari dua sisi: sisi pengguna (user side) dan sisi admin (admin side).

## Fitur

### User Side
- Pendaftaran dan Login
- Katalog Produk
- Detail Produk
- Keranjang Belanja
- Pembayaran

### Admin Side
- Login Admin
- Manajemen Produk
- Manajemen Pesanan
- Manajemen Pengguna

## Package

- React.js
- Axios
- React Router
